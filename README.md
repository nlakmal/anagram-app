## About Project

Given a list of words from an English dictionary and a word, return a set of words in the dictionary that are
anagrams of the word.

## Installation
1. Clone repository
2.  Rename .env.example to .env and add the correct details.
3. Run `composer install`
4. Run `php artisan serve`
4. Open `http://127.0.0.1:8000`

## Test

Run `php artisan test`
